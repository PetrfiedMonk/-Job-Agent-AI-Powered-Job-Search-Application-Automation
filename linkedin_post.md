I got tired of sending the same resume to 50 companies and hearing nothing.

So I built an AI agent that does the whole thing for me.

Here's what it does while I sleep:

→ Reads my Obsidian knowledge base and mines it for skills and projects I forgot I had
→ Searches Indeed, LinkedIn, ZipRecruiter, and Glassdoor simultaneously
→ Scores every job posting 0–100 for fit and salary potential before touching it
→ Rewrites my resume from scratch for each role — mirroring the exact language of the job description
→ Opens a browser, fills out the application form, uploads the tailored resume, and submits

Every. Single. Application. Customized. Automated.

The insight that drove this: volume matters in job hunting, but so does relevance. Most people sacrifice one for the other. This does both.

I built it because I already built SeekBridge.ai — a web tool for the same problem. But a web UI isn't an agent. I wanted something that could run fully autonomous, learn from my entire body of work, and get smarter the more I use it.

The whole stack: Python + Claude API + Playwright + jobspy + SQLite. Open source on GitHub.

It's not magic. The forms still break sometimes. LinkedIn's anti-bot detection is annoying. But it already fills applications faster than I can and the resumes it generates consistently score higher on ATS tools than my original.
